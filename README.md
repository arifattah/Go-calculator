# Go-calculator
Program sederhana berbasis terminal untuk melakukan operasi matematika dasar menggunakan bahasa Go.

# Fitur
- Penjumlahan (+)
- Pengurangan (-)
- Perkalian (*)
- Pembagian (/)

# Cara Menjalankan
go run main.go

# Contoh Hasil Program
=== Go Calculator ==
Masukkan angka pertama: 8
Masukkan operator (+, -, *, /): /
Masukkan angka kedua: 2
Hasil: 8 / 2 = 4
