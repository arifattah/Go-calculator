# Go-calculator
Program sederhana berbasis terminal untuk melakukan operasi matematika dasar menggunakan bahasa Go.

# Fitur
- Penjumlahan (+)
- Pengurangan (-)
- Perkalian (*)
- Pembagian (/)

# Cara Menjalankan
go run main.go

# Contoh Hasil Program
=== Go Calculator == <br>
Masukkan angka pertama: 8 <br>
Masukkan operator (+, -, *, /): / <br>
Masukkan angka kedua: 2 <br>
Hasil: 8 / 2 = 4
